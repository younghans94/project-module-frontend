# Spring 动画

基于 react-spring 的动画组件示例。

## 代码演示

### 基础动画

```tsx
import { Animate } from '../../src/basic/animate/spring/Animate';

export default () => <Animate />;
```

### 过渡动画

```tsx
import { SlideOverLayer } from '../../src/basic/animate/spring/Animate';

export default () => <SlideOverLayer />;
```
